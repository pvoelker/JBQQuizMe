Simple quizzing game for kids based on 10-point Junior Bible Quizzing questions. Works great for young learners as there is an option for the questions and possible answers to be read to them.

If a wrong answer is selected, it is disabled. Another answer can be selected until the correct one is found. So the game is a teaching and practice tool.

Animations for correct and wrong answers help keep the kids engaged. As correct answers are given, a candle glows brighter and brighter until it is 'lit'.

If you are interested in Junior Bible Quizzing, checkout: https://kidmin.ag.org/ministries/JBQ/overview
