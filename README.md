# JBQQuizMe

Simple quizzing game for kids based on Junior Bible Quizzing (JBQ) 10-point questions

## Building

dotnet publish -f net6.0-windows10.0.19041.0 -c Release /p:RuntimeIdentifierOverride=win10-x86
